# Full-Stack Job Application Tracker

A full-stack web app to track job applications end-to-end — built to demonstrate real-world CRUD workflows, authentication, clean code structure, and simple analytics.

## Features
- User signup/login (JWT authentication)
- Create, edit, delete, and view job applications
- Track: company, role, status (Applied / Interview / Offer / Rejected)
- Notes + follow-ups (keep context and next steps in one place)
- Filter + sort applications (by status, date, company, role, etc.)
- Simple analytics
  - Applications per week
  - Conversion rate (e.g., Interview ÷ Applied)

## Tech Stack
- Frontend: Next.js + TypeScript
- Backend: Node.js (Express) + TypeScript
- Database: PostgreSQL + Prisma ORM
- Auth: JWT
- Deployment: Vercel
